# **App Name**: JUST MARKETS

## Core Features:

- User Authentication: Secure registration and login for both traders and followers.
- Trader Profile Management: Allows expert traders to create and manage their profiles, display trading strategies, and show verified performance metrics.
- Follower Trader Discovery: Enables followers to browse and filter available traders, view their historical performance, risk scores, and success rates.
- AI-Powered Trader Recommendation Tool: An AI tool that suggests suitable traders to followers based on their declared risk tolerance, investment goals, and preferred asset classes, using a comprehensive analysis of market conditions and trader history.
- Copy Trading Configuration: Allows followers to set up and configure their copy trading parameters, including investment amounts, preferred assets, and stop-loss limits.
- Real-time Trade Execution: Automated system for replicating trades from chosen master traders to follower accounts in real-time.
- Performance Dashboard: Provides users with a personalized dashboard to track their portfolio's performance, view copied trade history, and monitor active investments.

## Style Guidelines:

- The visual design draws inspiration from the trustworthiness and structured nature of financial markets. The scheme is light for clarity and professionalism.
- Primary color: A sophisticated and stable muted blue-purple (#555578) to convey reliability and expertise. (HSL: 240, 40%, 35%)
- Background color: A very light, almost white, subtle blue-purple (#F1F1F8) ensures readability of complex financial data while maintaining brand consistency. (HSL: 240, 15%, 95%)
- Accent color: A fresh, clean aqua-cyan (#66BBCC) for interactive elements, buttons, and highlights, providing contrast and guiding user attention. (HSL: 210, 60%, 60%)
- Body and headline font: 'Inter' (sans-serif) is chosen for its modern, highly legible, and objective characteristics, essential for displaying data-heavy financial information clearly.
- Use a set of clean, minimalist line icons for navigation, data visualization, and key actions to maintain a professional and uncluttered aesthetic.
- A grid-based, organized layout with ample whitespace to ensure financial data is digestible and user interfaces feel intuitive and uncluttered across all device sizes.
- Subtle, smooth transitions for data updates, chart rendering, and navigational changes to enhance user experience without causing distraction or delays.
